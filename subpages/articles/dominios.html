Domínios

Domínios são endereços virtuais únicos que permitem aos usuários localizar sites sem memorizar endereços IP, conectando-os a servidores.

By Carlos Eduardo, Renato Rodrigues e Matheus Mendes

3 de Outubro de 2024

Um domínio seria o equivalente a um endereço físico para o seu site. Ele ajuda os usuários a encontrar seu site com facilidade, sem ter que memorizar seu endereço de protocolo de internet (IP). Nomes de domínio que consistem em um nome e uma extensão são parte fundamental da infraestrutura da internet. É o endereço virtual de um site também conhecido como URL, ele é utilizado para encontrar o seu site no meio de milhares de outras opções. O domínio vem logo após o www, é sempre cada domínio será único, nunca existindo dois domínios iguais.
Os domínios funcionam da seguinte forma, cada site tem dois elementos principais: um domínio e um servidor de hospedagem. Todos os domínios são ligados aos seus respectivos endereços de IP e apontam para o servidor específico que hospeda determinado site.
Quando um usuário insere um novo domínio no navegador, ele procura pelo endereço de IP associado através da rede global de servidores DNS (Sistema de Nomes de Domínio).
A seguir, o servidor com a informação sobre o endereço de IP retorna os dados ao navegador. O browser, por sua vez, solicita dados sobre o site ao serviço de hospedagem do domínio. Esse servidor armazena todos os dados do site, incluindo seus arquivos, seu banco de dados, e o seu código HTML. Assim que o host tiver enviado os dados de volta, o navegador irá convertê-los numa página que os usuários podem visitar.
